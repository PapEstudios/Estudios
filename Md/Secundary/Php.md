## Que es php?
    Es un lenguaje de codigo popular utilizado para backend y puede ser incrustado en el html 
    EJemplo:

```html
<!DOCTYPE html>
<html>
<head>
    <body>
    <?php
    echo "!hola, soy un script de PHP!",
    >
    </body>
</head>
</html>
```
